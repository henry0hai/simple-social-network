APIs

  • /api/make-friend 1. As a user, I need an API to create a friend connection between two email addresses.
  • /api/friends 2. As a user, I need an API to retrieve the friends list for an email address.
  • /api/get-common-friends 3. As a user, I need an API to retrieve the common friends list between two email addresses.
  • /api/subscribe 4. As a user, I need an API to subscribe to updates from an email address.
  • /api/block 5. As a user, I need an API to block updates from an email address.
  • /api/compose-message 6. As a user, I need an API to retrieve all email addresses that can receive updates from an email address.
